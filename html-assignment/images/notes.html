5 Unordered HTML Tags That I Have Learned

  • < ! DOCTYPE HTML > - Included at the top of the page, this is one of the most important pieces because it defines what type of document it will be.
  • Headings - Heading tags are used as a'title' and can be labeled h1,h2 and so forth.
  • Paragraphs - To add a pagraph tag, we insert the tags < p > < / p >.
  • Strong - We use this tag (< strong > < / strong >) to make the words in the text bold.
  • Highlight - When we want to highlight a word, we include the tag < mark > < / mark >.
